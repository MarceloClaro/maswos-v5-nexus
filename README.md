## 🚀 Sobre o Projeto

O **MASWOS V5 NEXUS** não é um mero script. Trata-se de uma rede neural autônoma mapeada sobre arquiteturas Transformers (Attention Mechanisms, Encoders, Validation Layers) operando 100% via **Model Context Protocol (MCP)** em ambientes como o OpenCode e Google Antigravity. 

Projetado para lidar com sobrecargas de produção intelectual, o sistema é dividido em **Poderosas Legiões de Agentes** voltadas para as searas Acadêmica e Jurídica.

## ⚔️ As Legiões (130+ Agentes Orquestrados)

- **🏛️ O Enxame Acadêmico (55 Agentes)**: Capaz de gerar papers Qualis A1. Batedores (Scrapers) raspam a CAPES e NCBI. Alquimistas (Data Scientists) manipulam CSVs do IBGE e Banco Mundial. Formadores (ABNT Makers) finalizam as margens da tese.
- **⚖️ O Enxame Jurídico (60 Agentes)**: Robôs de Peticionamento Acelerado e pesquisa jurisprudencial massiva na esteira do Supremo Tribunal Federal (STF/STJ) com validadores OAB Factual Logger.
- **👁️ Os Oráculos RAG (Estratégias de Contexto)**: O cérebro local atua utilizando 9 tipos de Retrieval-Augmented Generation, entre eles GraphRAG (para teses gigantescas combinadas com o `PageIndex`) e CRAG (Corrective-RAG contra alucinações bibliográficas severas).

## ⚙️ Implantação Rápida no Antigravity / OpenCode

### Pré-requisitos
- Python 3.10 ou Superior
- Uma interface devida que suporte MCP (Antigravity da Google ou VSCode modificado pelo OpenCode)

### Passo-a-passo (Deploy Autônomo)
1. **Clone esta Egrégora:**
   ```bash
   git clone https://github.com/MarceloClaro/maswos-v5-nexus.git
   cd maswos-v5-nexus
   ```

2. **Injetar Energias (Chaves de API):**
   Copie o arquivo mestre do ambiente.
   ```bash
   cp .env.example .env
   # Abra o .env e insira as suas chaves do OpenAI / NCBI
   ```

3. **Construção Modular (Bibliotecas):**
   ```bash
   pip install -r requirements.txt
   ```

4. **Binding Espacial (MCP Link):**
   Execute a rotina final para espelhar as portas na sua arquitetura mestra:
   ```bash
   python install_maswos_to_antigravity.py
   ```

## 👑 Modo Deus (End-game Pipeline)

Com as legiões unificadas, você não precisa conversar com os Agentes. Você **Ordena** que a rede inteira crie um artefato final massivo assíncrono.
Execute na raiz da sua aplicação para forjar documentos de mais de 100 páginas autonomamente (Apenas deixe os Scrapers, o RAG e o Cross-Validator fazerem o trabalho).
```bash
python run_thesis_god_mode.py
```
